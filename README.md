# CRUD USING REACT, NODE, MONGODB AND EXPRESS.


How to Setup Project Client Side:

1. git clone
2. cd CRUD
3. cd client
4. npm install
5. npm run dev


How to Setup Server Side:
1. cd Server
2. npm init -y
3. npm start
